メモ

ゲームタイトル「Knight of Run （ナイト・オブ・ラン）」
　略してナイラン。ないらん。内乱。はっ！アヴァリス帝国の内乱…！

= Game =
・同時に消すほどスコアが上がるスコアシステム欲しい

・課金（敵を倒した経験値で当たり判定の強化などの緩和機能）実装したい
・課金したらスコアアップにデメリット
・マウスの当たり判定の幅どうしよっかなー。
・画面の背景が上から下に流れるようにしたい。（疾走感的な）
・ゲームオーバーの回数で死亡時のコメント変わるようにしたい。
・障害物とかほしいよね

= System =
・イベント処理は別ファイル化したい
・シナリオをプログラム内にダラダラ書くのなんかなぁ
・敵、マウスカーソルの当たり判定もメソッド化したい
・図形描画と文字メソッド化したい
・変数の使い方ぐちゃぐちゃだから最低限予約済み一覧表作りたい
============================================
不具合

- 拡張関数GTで文字を出力すると「”」が末尾についてしまう。

- #6で１つのイベント内で８行以上のメッセージをプログラムすると２回目の呼び出し時に落ちる。

- 何故か特定の文を#6で出力するとゲーム起動前に落ちる。
　（対象文）
	GTx,y,"急ぎ城から脱出したあなたを守るのは";
	改善＞GTx,y,"いそぎ城から脱出したあなたを守るのは";

	GTx,y,"あーあ。死んじゃった。";
	改善＞GTx,y,"死んでしまったね…。";

	GTx,y,"　　- キーを押して時を戻す -";
	改善＞GTx,y,"　　＝　何かボタンを押して時を戻す　＝";

	GTx,y,"お願いします、どうかオーサスまで頑張ってください！";
	改善＞GTx,y,"お願いします、どうかオ-サスまで頑張ってください！";

	アダマス
	改善＞ADAMAS

	マウス
	改善＞まウス

　　・「ー」全角の伸ばし棒はバグるっぽい？（あーあ、キー、オーサス）
　　・「急ぎ」「アダマス」「マウス」は謎。
